Справочник предназначен для хранения описания всех форматов штрих-кодов (ШК), используемых в системе. Содержит три предопределенных, необходимых для работы, формата штрих-кодов; список может быть расширен пользователями.

Справочник вызывается из панели навигации подсистемы «НСИ».

Если штрих-код используется для маркировки весовых товаров, то для него актуально хранение количества товара в самом штрих-коде. Для такого формата ШК необходимо проставить флаг «Весовой» и указать множитель количества, на который при разборе ШК будут умножаться разряды, в которых указано количество.
